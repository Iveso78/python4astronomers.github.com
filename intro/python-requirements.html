Python requirementsÂ¶

In order to follow along with the examples to be presented in the workshops your must have at least the following installed in your working environment:

  • Python 2.6 or 2.7 [1]
  • NumPy >= 1.3
  • SciPy >= 0.8.0
  • Matplotlib >= 1.0.0
  • asciitable
  • APLpy
  • ATpy
[1]Python 3.x is the “next generation” Python but for astronomy analysis Python 2.x is still the best choice.
